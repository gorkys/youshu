# 有数 App - 产品需求与开发方案（完整版）

## 一、产品定位

**产品名：** 有数
**Slogan：** 心中有数，遇事不怵

**核心价值：**

- 记录物品位置
- 管理有效期
- 避免遗忘与浪费
- 支持 AI 自然语言搜索

------

## 二、核心原则（必须遵守）

1. 录入 ≤ 10 秒完成
2. 不强制填写信息
3. AI 只做建议，不做强制

------

## 三、核心功能模块

### 1. 拍照录入（核心功能）

#### 流程：

1. 点击拍照
2. 拍摄物品
3. AI 自动识别
4. 点击保存

------

#### 字段设计

| 字段   | 必填 | 说明     |
| ------ | ---- | -------- |
| 图片   | ✅    | 拍照获取 |
| 名称   | ✅    | AI识别   |
| 分类   | ❌    | AI建议   |
| 位置   | ❌    | 默认最近 |
| 数量   | ❌    | AI建议   |
| 有效期 | ❌    | OCR识别  |
| 价格   | ❌    | AI估算   |

------

### 2. 物品管理

- 状态：
  - 在用
  - 已用完
  - 已丢弃
- 支持：
  - 编辑
  - 删除
  - 标记完成

------

### 3. 搜索系统

#### 普通搜索：

- 名称
- 分类
- 位置

#### AI搜索：

示例：

- “冰箱里有什么快过期的”
- “我的充电器在哪”

------

### 4. 到期提醒

- 7天内提醒
- 本地通知

------

### 5. 分类系统

双维度：

- 分类（食品/药品）
- 位置（厨房/冰箱）

------
