# 一、整体设计语言（Design System）

## 1️⃣ 视觉关键词

- 主色：橙色渐变（品牌感）
- 风格：卡片式 + 圆角大弧度
- 层级：明显分层（卡片浮起）
- 情绪：温暖 / 轻松 / 生活化

------

## 2️⃣ 色彩规范

```
Primary Gradient:
#FF8A00 → #FFA940

Background:
#F6F7FB（浅灰）

Card:
#FFFFFF

Text:
#1F1F1F（主）
#8C8C8C（次）

Accent:
#FF4D4F（提醒/过期）
#52C41A（正常）
```

------

## 3️⃣ 圆角与阴影

```
全局圆角：16dp
卡片圆角：20dp
按钮圆角：24dp

阴影：
elevation: 4dp
颜色：rgba(0,0,0,0.06)
```

------

## 4️⃣ 动效规范

- 页面切换：淡入 + 上浮（200ms）
- 按钮点击：缩放 0.96
- 卡片出现：渐变透明 + 位移

------

# 二、首页（核心入口重构）

⚠️ 不要像示例那样“信息爆炸”，你的首页必须更克制

------

## 2.1 首页结构

```
顶部渐变区（品牌 + 搜索）

核心卡片（关键功能）
- 快速拍照录入（主入口）
- 即将过期提醒

分类/空间入口（次级）

物品列表（最近/全部）

底部导航
```

------

## 2.2 顶部区域（借鉴但简化）

```
背景：橙色渐变

内容：
[有数]
[搜索框（支持AI搜索）]
```

👉 不要复杂说明文字，保持干净

------

## 2.3 核心卡片（最重要）

```
大卡片（渐变背景）

标题：快速记录物品
副标题：拍一下就记住

[拍照按钮（超大）]
```

👉 这个卡片 = 产品灵魂

------

## 2.4 到期提醒卡片

```
白色卡片

标题：即将过期
内容：3件物品

[查看]
```

------

# 三、拍照录入页（重点中的重点）

## 3.1 相机界面

```
全屏预览

底部：
[拍照按钮（大圆）]
[相册入口]
```

------

## 3.2 拍照后确认页（核心体验）

👉 风格：卡片浮层 + 半透明背景

------

## 3.3 UI结构

```
背景：拍摄图片（模糊）

中间卡片（白色浮层）：

[图片预览]

名称（大字体）
分类标签（小胶囊）

位置（带图标）

[保存按钮（橙色渐变大按钮）]

—— 折叠区 ——
数量
有效期
价格
备注
```

------

## 3.4 关键交互

### 保存逻辑

- 点击直接保存
- 动效：卡片缩小 → 返回首页

------

### AI标识

- 小标签：
   👉 “AI识别”

------

### 折叠区

- 默认收起
- 滑动展开

------

# 四、物品列表页

## 4.1 列表卡片设计

```
左：图片（圆角）

右：
名称
位置（灰色）
有效期（红/绿提示）

右侧：
状态标签
```

------

## 4.2 风格

- 卡片间距大
- 留白多
- 不要密集信息

------

# 五、详情页（滑动核心）

## 5.1 顶部图片区（关键）

```
全宽图片

支持左右滑动（切换物品）
```

------

## 5.2 滑动体验（必须明确）

👉 类似“相册”

### 行为：

- 左滑 → 下一个物品
- 右滑 → 上一个物品

------

### UI提示

- 底部小点（分页指示）
- 或轻微提示动画

------

## 5.3 信息区

```
名称（大）

分类 + 位置（标签）

数量
有效期
价格

备注

评价
```

------

## 5.4 操作区

```
[标记已用完]
[编辑]
[删除]
```

👉 主按钮必须突出

------

# 六、底部导航（简化版）

```
首页
分类
拍照（中间凸起）
搜索
我的
```

👉 中间拍照按钮要突出（类似相机App）导航按钮应该是悬浮以及毛玻璃的效果

------

# 七、设计重点总结（给AI看的）

## 必须做到

### 1️⃣ 强视觉引导

- 橙色渐变 = 品牌核心

------

### 2️⃣ 卡片化

- 所有信息用卡片承载

------

### 3️⃣ 大按钮

- 拍照 / 保存必须突出

------

### 4️⃣ 留白

- 不要堆信息

------

## 必须避免

❌ 信息密集（像政务APP）
 ❌ 小按钮难点击
 ❌ 表单式录入

------

  - Card
  - LazyColumn
  - HorizontalPager（滑动详情）

- Accompanist Pager / ViewPager2

------

```
)
```
